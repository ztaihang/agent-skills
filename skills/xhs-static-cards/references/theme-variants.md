# Theme Variants（同族内切换）

生成 HTML 前结合 **taste-skill** 与文案 mood 选择 `content.json` 的 `theme` 字段。

## skill-list 族

| theme | 适合内容 | 特征 |
|-------|----------|------|
| `soft-pink` | Skill 推荐、工具清单 | 淡粉底、居中卡片（默认，接近常见小红书清单体） |
| `clean-white` | 专业工具、B2B | 白底、细线框、低饱和 |
| `tool-slate` | 开发者向 Codex/Cursor | 浅灰底、偏 terminal 标签色 |

同族内只允许改 **配色、圆角、装饰**；不得改字段结构（编号、标签、场景行必须保留）。

## vibecoding 族（P3）

| theme | 特征 |
|-------|------|
| `dark-terminal` | 深底、等宽字体、步骤编号 |
| `slate-minimal` | 深灰、大标题、步骤列表 |
